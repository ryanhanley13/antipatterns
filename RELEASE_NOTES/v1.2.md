First release with the deterministic density scanner and a real release pipeline. If you ran an earlier build, update to get `scan.py` and batch pattern additions.

## What's new

- **`scan.py` — the density sweeper.** The skill's thesis is that AI tells are about *density*, not banned words. `scan.py` makes that measurable: it reads `ANTIPATTERNS.md` as its source of truth and counts every Tier 1/2/3 word, exact phrase, and em-dash, then reports per-paragraph density so you can see where tells cluster instead of eyeballing it. Lexical only — structure (antithesis, triplet rhythm) stays the model's job. Flags: `--json` for machine-readable output, `--strict` for a nonzero exit (CI / pre-commit), `--allow` to suppress a word that's part of your voice.
- **`add_pattern.py --apply`** — add patterns in batch from a JSON spec, the way to apply the skill's own proposals without retyping them. Every edit is round-trip validated before writing, so a malformed addition is rejected and the catalog never gets corrupted. `--dry-run` previews; the interactive mode still works.
- **Catalog fixes** — the scanner exposed two inconsistencies, both corrected: `critical` was missing from the Tier 3 adjectives, and the tier overview disagreed with the authoritative §11 on `leverage` / `robust` / `navigate` / `foster`.
